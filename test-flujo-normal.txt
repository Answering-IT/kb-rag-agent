=== INSTALACIÓN ===

Paso 1: Crear cuenta
$ curl -X POST https://api.cloudstream.io/signup

Paso 2: Obtener API Key
$ cloudstream auth login

Paso 3: Instalar CLI
$ npm install -g cloudstream-cli

Paso 4: Configurar proyecto
$ cloudstream init my-project

Paso 5: Desplegar
$ cloudstream deploy --env production
