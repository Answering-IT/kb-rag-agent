🎯 RETRIEVE TOOL USAGE (from Strands tests):

Correct filter format:
{
    "andAll": [
        {"equals": {"key": "tenant_id", "value": "1001"}},
        {"equals": {"key": "partition_key", "value": "t1001_p165"}}
    ]
}

Example in code:
result = agent.tool.retrieve(
    text="What are the requirements?",
    retrieveFilter={
        "andAll": [
            {"equals": {"key": "tenant_id", "value": "1001"}},
            {"equals": {"key": "partition_key", "value": "t1001_p165"}}
        ]
    },
    score=0.5,
    numberOfResults=10
)
